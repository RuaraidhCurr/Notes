Similar to Java apex allows for conditional if & else statements e.g.

```
Integer x, sign; 
// Your code 
if (x <= 0) { 
	if (x == 0) { 
		sign = 0; 
	} else { 
	sign = -1; 
	} 
}
```

Or Also (outcome of both above and below are the same)

```
Integer x, sign; // Your code if (x <= 0) if (x == 0) sign = 0; else sign = -1;
```
